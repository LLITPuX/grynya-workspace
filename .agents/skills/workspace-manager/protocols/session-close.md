# Протокол: Закриття сесії (Session Close)

## Призначення
Фіксація досягнутого прогресу, отриманого досвіду та виправлених помилок. Виконується **перед завершенням** кожної робочої сесії для кожного скілу, що був використаний.

## Алгоритм

1. **Запустити скрипт:**
   ```bash
   node .agents/skills/workspace-manager/scripts/session-log.js close <назва-скілу>
   ```

2. **Заповнити анкету в JSON:**
   Відкрити `.agents/skills/<назва-скілу>/logs/journal.json` та відредагувати **останній об'єкт** у масиві:

   - **`status`**: "Завершено", "В процесі" або "Перервано"
   - **`summary`**: Чіткий опис того, що було зроблено (українською)
   - **`errors`**: Список помилок, які виникли (навіть якщо ви їх виправили)
   - **`lessons`**: Найважливіша частина. Опишіть проблему та рішення (`problem` / `solution`)
   - **`next_steps`**: Що потрібно зробити наступного разу

3. **Зберегти та закомітити:**
   Журнал має бути частиною Git-репозиторію.

> [!IMPORTANT]
> Будьте чесними в описі помилок. Це ваш "бортовий журнал", який допоможе вам (або іншому агенту) уникнути тих самих граблів у майбутньому.
