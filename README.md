# Grynya Workspace

Робочий простір AI-агента Grynya — модульна екосистема для автоматизації розробки.

## Архітектура та Інфраструктура

- **Docker-first**: Весь воркспейс та sidecars працюють у контейнерах.
- **FalkorDB**: Графова база даних для зберігання структури проекту та історії коммітів.
- **Firecrawl**: Розумний скрапер для дослідження зовнішніх ресурсів.

## Структура

| Каталог | Призначення |
|---------|-------------|
| `.agents/skills/` | Скіли агента (логіка, скрипти, протоколи) |
| `.agents/workflows/` | Автоматизовані воркфлоу |

## Основні функції

- **Git-to-Graph Sync**: Автоматична синхронізація структури репозиторію та історії в FalkorDB.
- **Session Logging**: Журналювання кожної сесії в `journal.json` кожного скілу.

## Скіли

- **workspace-manager** — управління робочим простором (ініціалізація, інфраструктура, створення скілів).
