# Grynya Workspace

Робочий простір AI-агента Grynya — набір скілів, протоколів та інструментів для автоматизації розробки.

## Структура

| Каталог | Призначення |
|---------|-------------|
| `.agents/skills/` | Скіли агента |
| `.agents/workflows/` | Автоматизовані воркфлоу |

## Скіли

- **workspace-manager** — управління робочим простором (ініціалізація, Firecrawl, створення скілів)

## Мова

Уся документація ведеться **українською мовою**.
